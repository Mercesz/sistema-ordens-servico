# Sistema de Ordens de Serviço

Projeto Front-End desenvolvido em React para gerenciamento de ordens de serviço.

## Funcionalidades
- Criar nova ordem
- Listar ordens
- Atualizar status (Aberta, Em andamento e Finalizada)
- Filtro por status
- Integração com API (JSON Server)

## Tecnologias
- React
- JavaScript
- Axios
- JSON Server
- CSS

## Como rodar o projeto
```
  npm install
  npm run dev
  npm run server
```
